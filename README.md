# GigHub API

Admission Number: C027-01-0854/2024

This project is a REST API built using FastAPI.

Features

- List all gigs
- Filter gigs by category
- Filter gigs by minimum and maximum budget
- Search gigs by title
- View one gig
- Create a gig
- Update budget and status
- Delete a gig

Run

```bash
uvicorn main:app --reload
```

Swagger Documentation

```
http://127.0.0.1:8000/docs
```
